## :zap: Description 📃

This is a Bow And Arrow Game made using HTML, CSS and Javascript


## :zap: How to run it? 🕹️

- Click Start to play.
- Mouse Click or press any key to fire the arrow!
- Goal: Hit the Target as many times as you can for the highest score.
